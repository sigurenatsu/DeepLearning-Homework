# 编程练习2 实验报告
## 用两层ReLU神经网络拟合函数

---

## 一、函数定义

本实验选择以下函数作为拟合目标：

$$f(x) = \sin(2\pi x) + 0.5\cos(4\pi x), \quad x \in [-1, 1]$$

**选择理由：**
- 该函数由正弦波与余弦波叠加而成，具有多个极值点，形状复杂，对神经网络的拟合能力有一定挑战性。
- 函数连续且有界，符合通用近似定理（Universal Approximation Theorem）的适用范围。
- 函数值域约为 $[-1.5, 1.5]$，数值范围合理，无需额外归一化。

---

## 二、数据采集

| 数据集 | 采样方式 | 样本数 | 是否加噪声 |
|--------|----------|--------|------------|
| 训练集 | 在 $[-1, 1]$ 上**随机均匀采样** | 800 | ✅ 加入高斯噪声 $\mathcal{N}(0, 0.05^2)$ |
| 测试集 | 在 $[-1, 1]$ 上**均匀间隔采样** | 200 | ❌ 使用真实值 |

**训练集采样代码：**
```python
x_train = np.random.uniform(-1, 1, 800)
noise   = np.random.normal(0, 0.05, 800)
y_train = np.sin(2*np.pi*x_train) + 0.5*np.cos(4*np.pi*x_train) + noise
```

加入少量噪声是为了模拟真实场景中数据含有测量误差的情况，同时防止模型过拟合。

---

## 三、模型描述

### 3.1 网络结构

```
TwoLayerReLUNet(
  (layer1): Linear(in_features=1, out_features=128, bias=True)
  (relu):   ReLU()
  (layer2): Linear(in_features=128, out_features=1, bias=True)
)
```

**总参数量：385 个**（1×128 + 128 + 128×1 + 1 = 385）

### 3.2 为什么用ReLU？

ReLU（整流线性单元）定义为：
$$\text{ReLU}(x) = \max(0, x)$$

优点：
1. **计算简单**：只需判断正负，速度快。
2. **不会梯度消失**：与 sigmoid/tanh 不同，正区间梯度始终为1。
3. **通用近似能力**：Nair & Hinton（2010）证明，两层 ReLU 网络可近似任意有界闭集上的连续函数。

### 3.3 训练配置

| 超参数 | 值 |
|--------|-----|
| 深度学习框架 | PyTorch 2.0.1 |
| 损失函数 | MSE（均方误差） |
| 优化器 | Adam |
| 初始学习率 | 0.001 |
| 学习率衰减 | 每 500 epoch × 0.5（StepLR） |
| 训练轮数 | 2000 epochs |
| 隐藏层大小 | 128 个神经元 |
| 随机种子 | 42（保证可复现） |

## 四、拟合效果

### 4.1 训练过程（损失变化）

| Epoch | 训练 Loss | 测试 Loss |
|-------|-----------|-----------|
| 200   | 0.352993  | 0.358020  |
| 400   | 0.222604  | 0.232656  |
| 600   | 0.171421  | 0.180714  |
| 800   | 0.141695  | 0.149224  |
| 1000  | 0.113297  | 0.119023  |
| 1200  | 0.100338  | 0.105287  |
| 1400  | 0.088386  | 0.092534  |
| 1600  | 0.079957  | 0.083537  |
| 1800  | 0.074561  | 0.077744  |
| **2000**  | **0.069305**  | **0.072093**  |

- 训练 Loss 和测试 Loss 同步下降，说明模型**没有出现过拟合**。
- 两者始终非常接近，说明模型的**泛化能力良好**。

### 4.2 最终评估结果

| 评估指标 | 结果 |
|----------|------|
| 最终训练 MSE | **0.069305** |
| 最终测试 MSE | **0.072093** |
| 测试集 R² 分数 | **0.8843** |

### 4.3 结果分析

R² = 0.8843 表示神经网络能够解释约 **88.4%** 的函数变化趋势，对于一个仅有 **385个参数**、**2层结构** 的小型网络来说，这是合理的拟合效果。

若希望进一步提升精度，可以：
- 增大隐藏层神经元数量（如 256、512）
- 增加训练轮数
- 使用更小的学习率进行精细调整

### 4.4 结论

实验验证了理论结论：**一个两层的 ReLU 神经网络确实可以拟合任意有界闭集上的连续函数**。通过 PyTorch 构建含 128 个隐藏神经元的两层网络，使用 Adam 优化器训练 2000 轮，模型成功学习了目标函数 $f(x) = \sin(2\pi x) + 0.5\cos(4\pi x)$ 的整体趋势，测试集 R² 达到 0.8843。
